잡헌터(구인구직 사이트)

6인 팀 프로젝트

잡헌터는 사용자 친화적 인터페이스를 기반으로 한 구직자와 기업을 연결하는 구인구직 플랫폼입니다. Spring 프레임워크를 활용하여 개발되었으며, 1:1채팅 상담 기능과 고객센터 게시판을 구현했습니다.

초기에는 일반 컨트롤러를 사용하여 모든 페이지를 구현하려 했으나, 실시간 상호작용의 필요성을 인식하고 아키텍처를 개선했습니다. 1:1 채팅 기능은 RESTful API와 비동기 폴링 기법을 결합한 접근 방식을 채택했습니다.

서버 측에서는 채팅 메시지를 저장하고 조회하는 엔드포인트를 구현했고, 클라이언트 측에서는 fetch API를 사용하여 주기적으로 새 메시지를 폴링하는 방식으로 실시간 채팅과 유사한 효과를 제공했습니다.

특히, setInterval 함수를 활용하여 5초마다 서버로부터 최신 채팅 목록을 가져오는 reloadChat 함수를 구현했습니다. 이를 통해 실시간성을 확보했습니다.

또한, 동적 UI 렌더링에 중점을 두어 데이터의 상태와 사용자 권한에 따라 조건부로 컴포넌트를 렌더링하는 방식을 채택했습니다. 이를 통해 사용자 맞춤형 인터페이스를 구현하여 UX를 크게 향상시켰습니다.

잡빙수(구인구직 사이트)

6인 팀 프로젝트

잡빙수는 신선한 UI/UX 디자인과 매칭 알고리즘을 제공하는 구인구직 플랫폼입니다. Spring 프레임워크를 활용하여 개발되었으며, 기업 페이지 관리, 기업 공고 관리, 지원자 관리를 구현했습니다.
